TextMate Theme
==============

An XCode 4 colorscheme similar to the one I use in Vim and used to use
in TextMate.


Usage
-----

Copy the **`TextMate.dvtcolorscheme`** file to **`~/Library/Developer/Xcode/UserData/FontAndColorThemes/`**.
